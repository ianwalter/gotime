# gotime
> A high resolution timer CLI for timing how long commands take to execute

## Installation


## Usage

Just run `gotime` and then your command, for example:

```console
❯ gotime echo Hello World!
Hello World!
Time Elapsed: 622.304µs
```

I <3 microseconds.
